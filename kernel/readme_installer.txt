[Information]
Windows installer and uninstaller files added to the kernel zip file.

[Files]
1.wininstaller_takaserv.bat
Stars server 'takaserv' windows installer.
Install a 'Stars Server' windows service with default options.
    Stars Server port(default): 6057
    Service name(default): takaserv
Please run this program with windows administrator privileges.

2.winuninstaller_takaserv.bat
Just remove the 'Stars Server' windows service 'takaserv'.
Please run this program with windows administrator privileges.
