Constructor

Usage:
StarsInterface stars = new StarsInterface(String nodeName, String svrHost, String keyFile, int svrPort);
StarsInterface stars = new StarsInterface(String nodeName, String svrHost, String keyFile);
StarsInterface stars = new StarsInterface(String nodeName, String svrHost);

Create new instance. It returns reference of instance.
stars
StarsInterface type instance.
nodeName
Node mame of client.
svrHost
Host name of STARS server. 'localhost' is used if omitted.
keyFile
Keyword file name. 'nodeName.key' is used if omitted.
svrPort
Port number of STARS. 6057 is used if omitted.

connect

Usage:
void stars.connect(); //throws StarsException

Connect to STARS server
stars
StarsInterface type instance which created by the constructor.

receive

Usage:
StarsMessage rt = stars.receive(int timeout); //throws StarsException

rt
StarsMessage type variable
stars
StarsInterface type instance which created by the constructor.
timeout
Time out in m second. If omitted, default value (500m second) is used.
