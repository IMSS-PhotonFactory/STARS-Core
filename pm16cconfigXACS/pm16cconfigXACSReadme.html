pm16cconfigXACS Application Client概要

2007-10-01

はじめに

ツジ電子製PM16C-04Xシリーズに「Auto-Change-Drive機能」が追加されました。

「Auto-Change-Drive機能」は、モータ毎にあらかじめ登録された速度変更条件と速度変更内容に基づきモータ動作中の速度変更が可能となる機能です。
詳細は、「速度変更ポイント予約機能（ツジ電子）」をご確認ください。

本Stars Application Clientプログラム「pm16cconfigXACS」は、「Auto-Change-Drive機能」のモータ毎の速度変更条件と速度変更内容を登録する機能です。

アプリケーションの起動

  perl pm16cconfigXACS [MotorNumber] [StarsServer] [-c PM16C-04X_StarsNodename] [-node StarsNodeNameMe] [-row NumberOfDataToEdit] [-h] 
    MotorNumber                    '0' is the default value. Set motor number to be displayed after start-up.
    StarsServer                    'localhost' is the default value. Set Stars Server's network name/address.
    -c PM16C-04X_StarsNodename     'pm16c4x' is the default value. Set nodename of the Stars I/O client controlling 'PM16C-04X'.
    -node StarsNodename            'pm16cconfigXACS' is the default value. Set nodename me.
    -row NumberOfDataToEdit        '128' is the default value. Set display rows you want to set, to save display speed or computer memory.
    -h option help:Show command line help.

※1 Function「RTE」選択時の注意事項

  モータのAccRate（加減速レート）の値によって、「Auto Change Drive機能」のRTE（加減速レート）選択時の値の有効範囲が決まります。
  モータのAccRateと「Auto Change Drive機能」のRTE選択時の有効値FVAL（LLL)の関係は以下の通りです。
  AccRateはStars Application Client「pm16cconfig」から変更できます。AccRateはRate Data(not Rate Code)で入力してください。
  「Auto Change Drive機能」はRate Codeでの入力になります。

関連コマンド

SetAutoChangeSpeedReady(オートチェンジスキャンセット／解除コマンド)

Usage:    SetAutoChangeSpeedReady 1
Synopsis: オートチェンジスキャンを実行可状態にします。
Reply:    @SetAutoChangeSpeedReady 1 Ok: 
Error:    @SetAutoChangeSpeedReady 1 Er: Reason 
Example: 
pm16c4x.th SetAutoChangeSpeedReady 1 (term1から送信) 
pm16c4x.th>term1 @SetAutoChangeSpeedReady 1 Ok: (受信) 
pm16c4x.th GetAutoChangeSpeedReady 
pm16c4x.th>term1 @GetAutoChangeSpeedReady Ok: 1 (受信) 

Usage:    SetAutoChangeSpeedReady 0
Synopsis: オートチェンジスキャンを実行可状態を解除します。
Reply:    @SetAutoChangeSpeedReady 0 Ok: 
Error:    @SetAutoChangeSpeedReady 0 Er: Reason
Example: 
pm16c4x.th SetAutoChangeSpeedReady 0 (term1から送信) 
pm16c4x.th>term1 @SetAutoChangeSpeedReady 0 Ok: (受信) 
pm16c4x.th GetAutoChangeSpeedReady 
pm16c4x.th>term1 @GetAutoChangeSpeedReady Ok: 0 (受信) 


GetAutoChangeSpeedReady(オートチェンジスキャン状態確認コマンド)

Usage:    GetAutoChangeSpeedReady
Synopsis: オートチェンジスキャンが実行可状態かを返します。
Reply(1): @GetAutoChangeSpeedReady Ok 1: (実行可状態の時）
Reply(2): @GetAutoChangeSpeedReady Ok 0: (実行可状態でない時） 
Error:    @GetAutoChangeSpeedReady Er: Reason 
Example: 
pm16c4x.th SetAutoChangeSpeedReady 1 (term1から送信) 
pm16c4x.th>term1 @SetAutoChangeSpeedReady 1 Ok: (受信) 
pm16c4x.th GetAutoChangeSpeedReady 
pm16c4x.th>term1 @GetAutoChangeSpeedReady Ok: 1 (受信) 
pm16c4x.th SetValue 100000 (thを100000へ移動、駆動開始) 


SetValue(絶対位置指定によるモータ移動コマンド)

Usage:    SetValue Value 
Synopsis: モータをValueの値まで動かします。
Reply:    @SetValue Value Ok: 
Error:    @SetValue Er: Reason 
Example: 
PM16C04.th SetValue 12345 (term1から送信) 
PM16C04.th>term1 @SetValue 12345 Ok: (受信) 


SetValueREL(現在位置からの相対値指定によるモータ移動コマンド)

Usage:    SetValue Value 
Synopsis: モータを現在位置からValueの値分動かします。 
Reply:    @SetValueREL Value Ok: 
Error:    @SetValueREL Er: Reason 
Example: 
PM16C04.th SetValueREL 1000 (term1から送信) 
PM16C04.th>term1 @SetValueREL 1000 Ok: (受信) 


Standby(複数モータ同期スタート準備コマンド)

Usage:    Standby 
Synopsis: 同時駆動のために全チャンネルをStandby状態にする。 
Reply:    @Standby Ok: 
Error:    @Standby Er: Reason 
Example: 
pm16c4x.th SetAutoChangeSpeedReady 1 (term1から送信) 
pm16c4x.th>term1 @SetAutoChangeSpeedReady 1 Ok: (受信) 
pm16c4x.th GetAutoChangeSpeedReady
pm16c4x.th>term1 @GetAutoChangeSpeedReady Ok: 1 (受信) 
pm16c4x.dth1 SetAutoChangeSpeedReady 1 
pm16c4x.dth1>term1 @SetAutoChangeSpeedReady 1 Ok: (受信) 
pm16c4x.dth1 GetAutoChangeSpeedReady 
pm16c4x.dth1>term1 @GetAutoChangeSpeedReady Ok: 1 (受信) 
pm16c4x Standby (term1から送信) 
pm16c4x>term1 @Standby Ok: (受信) 
pm16c4x.th SetValue 100000 (thを100000へ、まだ駆動は開始しない) 
pm16c4x.th>term1 @SetValue 100000 Ok: (受信) 
pm16c4x.dth1 SetValue 100000 (dth1を100000へ、まだ駆動は開始しない) 
pm16c4x.dth1>term1 @SetValue 100000 Ok: (受信) 
pm16c4x SyncRun (駆動開始)) 
pm16c4x>term1 @SyncRun Ok: (受信)

Synopsis: 同時駆動の開始。 
Reply:    @SyncRun Ok: 
Error:    @SyncRun Er: Reason
