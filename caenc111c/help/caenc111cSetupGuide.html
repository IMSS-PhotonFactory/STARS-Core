SETUP

Place the Program Files into the same Folder.

Place the program files of 'caenc111c' in the same folder.

Copy the stars keyfile to Stars Server.

Stars keyfile name is [Stars Nodename].key.
If you name the stars nodename of 'caenc111c' as 'caenc111c', copy the file 'caenc111c.key' to [Stars Server's folder]/takaserv-lib/.

CUSTOMIZE config.pl

'config.pl' is the user configuration file. Please modify it.
 

$::Server="{Write IP Address Of Stars Server}"

Set the IP Address(or Name) of the StarsServer to the $::Server.

[Examples]
$::Server= "localhost";

$::CAMAC_HOST="{Write IP Address Of CAEN CAMAC Crate Controller C111C}"

Set the IP Address(or Name) of the CAEN C111C CAMAC Crate Controller to the $::CAMAC_HOST.

[Examples]
$::CAMAC_HOST="192.168.11.153";

Configuration of CAEN C111C

require 'caenc111cmod.pl';

Including the CAEN C111C library. (Don't change.)

CAENC111CRegister("{Write caenc111c NodeName}","CAENC111C","1.0");

Registers the information for CAEN C111C.

1st parameter: caenc111c NodeName
    Specify the Stars Nodename for caenc111c. Use "$::Nodename" normally.

2nd(="CAENC111C") and 3rd(="1.0") parameters are the tags for future use. Don't change them.

[Examples]
CAENC111CRegister("$::NodeName","CAENC111C","1.0"); # Usually no needs to be changed.

Configuration of Kenetic Systems Model 3655-L1A

require 'ks3655l1amod.pl';

Including the Kenetic Systems Model 3655-L1A library.

If no needs to use the Kenetic Systems 3655-L1A module, comment out the line as
  #require 'ks3655l1amod.pl';

KINETICSYSTEMS3655L1ARegister("{Write KS3655-L1A NodeName}","{Write KS3655-L1A SlotNumber}","CAENC111C","1.0");

Registers the information of Kenetic Systems Model 3655-L1A.

If no needs to use Kenetic Systems Model 3655-L1A module, comment out the line as
  #KINETICSYSTEMS3655L1ARegister(...);

1st parameter: KS3655-L1A NodeName
    Specify the Stars Nodename to use the Stars Commands of Kenetic Systems Model 3655-L1A.
    It should be named as "$::Nodename.xxxx"

2nd parameter: KS3655-L1A SlotNumber
    Specify the Slot Number(1 - 23) the Kenetic Systems Model 3655-L1A has been inserted in.

3rd(="CAENC111C") and 4th(="1.0") parameters are the tags for future use. Don't change them.

[Examples]
KINETICSYSTEMS3655L1ARegister("$::NodeName.ks3655","23","CAENC111C","1.0");

Configuration of CAEN C257

require 'caenc257mod.pl';

Including the CAEN C257 library.

If no needs to use the CAEN C257 module, comment out the line as
  #require 'caenc257mod.pl';

@::CAENC257CounterName = ({Write list of Counter Names})

Set the name list for CAEN C257 Counter from 0 to 15(16 items).
These are used to construct the Stars Nodename for CAEN C257 Counters.

If no needs to use the CAEN C257 module, comment out the line as
  #@::CAENC257CounterName=(...);

[Examples]
@::CAENC257CounterName=qw(C01 C02 C03 C04 C05 C06 C07 C08 C09 C10 C11 C12 C13 C14 C15 C16);

CAENC257Register("{Write CAEN C257 NodeName}", "{Write CAEN C257 SlotNumber}", "{Write Stars Parent Name of CAEN C257 Counters}" , "CAENC111C","1.0");

Registers the information of CAEN C257.

If no needs to use the CAEN C257 module, comment out the line as
  #CAENC257Register(...);

1st parameter: CAEN C257 NodeName
    Specify the Stars Nodename to use the Stars CAEN C257 Controller Commands.
    It should be named as "$::Nodename.xxxx"

2nd parameter: CAEN C257 SlotNumber
    Specify the Slot Number(1 - 23) the CAEN C257 has been inserted in.

3rd parameter: Stars Parent Name of CAEN C257 Counters
    Used to construct the Stars CAEN C257 Counters NodeName to use the Stars CAEN C257 Counter Commands.
    If "$::NodeName" specified, "$::Nodename.C01" (See previous step for what is 'C01') is the Stars Nodename for CAEN C257 Counter 0.
    If "$::NodeName.c257" specified, "$::Nodename.c257.C01" is it for CAEN C257 Counter 0.

4th(="CAENC111C") and 5th(="1.0") parameters are the tags for future use. Don't change them.

[Examples]
CAENC257Register("$::NodeName.c257","22","$::NodeName","CAENC111C","1.0");
