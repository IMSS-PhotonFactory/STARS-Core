INTRODUCTION

'caenc111c' is the Stars I/O Client Program, which is developped to control the CAEN CAMAC Controller Model C111C (C111C).
 
Slot module controls via C111C are also available, for Kenetic Systems Model 3655-L1A and CAEN Module C257.

NAME

caenc111c     - Stars I/O Client Program for CAEN CAMAC Controller Model C111C

SYNOPSIS

perl caenc111c [StarsNodeName] [-d] [-h]

StarsNodeName    'caenc111c' is the default value.
-d    option debug: Show debug messages to the display.
-h    option help:Show command line help.
