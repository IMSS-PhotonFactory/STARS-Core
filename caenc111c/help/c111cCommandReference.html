General

Stars client name of CAEN CAMAC Controller Model C111C

Stars client name of CAEN CAMAC Controller Model C111C
In this Manual, 'caenc111c' is used for the client name of CAEN C111C.

How to execute the Stars Command of CAEN CAMAC Controller Model C111C

How to execute the Stars Command of CAEN CAMAC Controller Model C111C

Send the Stars message as following format

caenc111c CAEN_C111C_Stars_Command Command_Parameters

  • 'caenc111c' is the Stars client name for CAEN CAMAC Controller Model C111C.

  • CAEN_C111C_Stars_Command

  • Command_Parameters    (Only necessary)


Then reply arrives as following

caenc111c>test @CAEN_C111C_Stars_Command Command_Parameters Command_Message

  • 'test' is used, in this manual, as the Stars client name that sends the Stars message.

  • Command_Message
    • starts with 'Ok: ' : the Stars command executed with success.

    • starts with 'Er: ' : the Stars command did not executed because of command nonexistent, command paramarer invalid, etc.

    • starts with 'Ng: ' : the Stars command executed but failed.

Error General

Error General

  1. Destination client name nonexistent :
    • (send)  caenc111 hello
      (reply)  System>test @hello Er: caenc111 is down.

  2. Command nonexistent :
    • (send)  caenc111c RunDataWayInit
      (reply)  caenc111c>test @RunDataWayInit Er: Bad Command.

  3. Command parameter missing or invalid :
    • (send)  caenc111 RunDatawayInit 1
      (reply)  caenc111c>test @RunDatawayInit 1 Er: Bad Parameter.

  4. Command executed but network error occured :
    (Usually not happens. But if it's happened, stop 'caenc111c' Program, restart CAMAC Controller CAEN C111C. then start 'caenc111c')
    • (send)  caenc111c RunDatawayInit
      (reply)  caenc111c>test @RunDatawayInit Er: -4
