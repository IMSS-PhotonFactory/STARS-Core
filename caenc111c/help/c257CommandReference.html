General

About CAEN Module C257 contoller commands

About CAEN Module C257 contoller commands
CAEN Module C257 has 2 types of command system, controller commands and counter commands.
Controller command system has the whole functions(including the counter command functions) of CAEN Module C257.
Be careful, the Stars client name for controller commands and that for counter commands are different.

Stars client name of CAEN Module C257 controller commands

Stars client name of CAEN Module C257 controller commands
In this Manual, 'caenc111c.c257' is used for the client name for CAEN Module C257 controller commands.

How to execute the Stars Command of CAEN Module C257 controller

How to execute the Stars Command of CAEN Module C257 controller

Send the Stars message as following format

caenc111c.c257 CAEN_C257_Controller_Stars_Command Command_Parameters

  • 'caenc111c.c257' is the Stars client name for CAEN Module C257 controller commands.

  • CAEN_C257_Controller_Stars_Command

  • Command_Parameters    (Only necessary)


Then reply arrives as following

caenc111c.c257>test @CAEN_C257_Controller_Stars_Command Command_Parameters Command_Message

  • 'test' is used, in this manual, as the Stars client name that sends the Stars message.

  • Command_Message
    • starts with 'Ok: ' : the Stars command executed with success.

    • starts with 'Er: ' : the Stars command did not executed because of command nonexistent, command paramarer invalid, etc.

    • starts with 'Ng: ' : the Stars command executed but failed.

Error General

Error General

  1. Destination client name nonexistent :
    • (send)  caenc111c.c257c hello
      (reply)  System>test @hello Er: caenc111c.c257c is down.

  2. Command nonexistent :
    • (send)  caenc111c.c257 hell
      (reply)  caenc111c.c257>test @hell Er: Bad Command.

  3. Command parameter missing or invalid :
    • (send)  caenc111c.c257 SetLAMEnable
      (reply)  caenc111c.c257>test @SetLAMEnableEr: Bad Parameter.

  4. Command executed but network error occured :
    (Usually not happens. But if it's happened, stop 'caenc111c' Program, restart CAMAC Controller CAEN C111C. then start 'caenc111c')
    • (send)  caenc111c.c257 SetLAMEnable 1
      (reply)  caenc111c.c257>test @SetLAMEnable 1Er: -4

Event Message List

_ChangedInhibitIsBusy

_ChangedInhibitIsBusy
Send by C257 controller when C257 Inhibit status has been changed, or 'flushdata','flushdatatome' command executed.

[Event Message Format]
_ChangedInhibitIsBusy 1  (when C257 input line has been inhibited)
_ChangedInhibitIsBusy 0  (when C257 input line inhibit has been cleared)

_ChangedLAMIsBusy

_ChangedLAMIsBusy
Send by C257 controller when LAM Line busy status has been changed, or 'flushdata','flushdatatome' command executed.

[Event Message Format]
_ChangedLAMIsBusy 1  (when LAM status is on)
_ChangedLAMIsBusy 0  (when LAM status is off)
